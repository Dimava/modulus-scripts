# Dimava Modulus Scripts

Personal gameplay scripts for Modulus, intended to run through `ScriptEngine`.

## Install

1. Install `ScriptEngine` into the game directory.
2. Copy this folder into `<GameDir>/Scripts/Dimava`.
3. Launch the game. `ScriptEngine` will discover the scripts automatically.

## Notes

- Per-script enable flags and keybindings are stored in `Scripts/ScriptEngine.cfg`.
- Several scripts define configurable hotkeys through the ScriptEngine UI or config file.
- `CameraHorizontalMirror.cs` mirrors relative to the current main camera yaw.
